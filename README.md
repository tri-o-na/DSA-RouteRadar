# RouteRadar

RouteRadar is a flight map route app that takes your source and destination airport, along with the flight date, to display the mapping, price, available airlines, layovers, and related information.

## Installation
To run RouteRadar, install the necessary dependencies:

```sh
pip install kivy kivymd kivy-garden.mapview
garden install mapview
```

## Usage
Run the application with:

```sh
python MapRoutes.py
```

## Features
- Search for flights between airports
- Display flight routes on a map
- Show ticket prices and available airlines
- Include layover details
- Include baggage details
- Filter by price, airline and layover

## GUI
- Used a python GUI called kivy to show the routes
